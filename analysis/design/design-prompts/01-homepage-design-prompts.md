# Homepage Design Prompts

Use together with 00-shared-brand-and-style.md.

## Prompt A - Homepage Desktop Design Example

Create a high-fidelity desktop homepage design concept for Rhino Inquisitor.

Structure from top to bottom must follow this order:

1. Header with logo, nav links (Blog, Topics, About), search input/icon, and a primary Start Reading button.
2. Hero section with short personal intro text and a complementary visual area.
3. Hero action row with two actions: Start Reading and Browse Topics.
4. Three-lane discovery section:
   - Lane 1: Featured long-form article card (large card with title, short excerpt, metadata).
   - Lane 2: Recent posts cards (three compact cards).
   - Lane 3: Topic hubs with chips and item counts.
5. Compact Active Projects rail with 2-3 project rows.
6. Newsletter/subscribe strip.
7. Footer.

Design intent:

- Modern editorial blog with crisp type and clear visual hierarchy.
- Strong first impression for the featured article.
- Discovery modules should be clearly differentiated but visually cohesive.
- Keep the page airy and readable; avoid visual clutter.

Framing:

- Desktop browser frame.
- Landscape composition.

## Prompt B - Homepage Mobile Design Example

Create a high-fidelity mobile homepage design concept for Rhino Inquisitor.

Structure from top to bottom must follow this order:

1. Mobile header with logo, menu icon, search icon.
2. Hero intro block with two CTA buttons.
3. Featured long-form article card.
4. Recent posts section with 3 stacked cards.
5. Topics section with chips or compact topic tiles.
6. Compact Active Projects list.
7. Newsletter/subscribe strip.
8. Footer.

Design intent:

- Single-column reading-first flow.
- Keep visual weight on content cards and headings.
- Maintain strong tap targets and comfortable spacing.
- Preserve the same visual language as desktop.

Framing:

- Smartphone portrait frame.
- Full page visible.
