# Shared Brand and Style Prompt

Paste this first before any page-specific prompt.

Create a high-fidelity website mockup image for Rhino Inquisitor, a technical blog about Salesforce B2C Commerce Cloud.

Core objective:

- Keep the page structure aligned to the provided wireframe.
- Modernize the visual design while preserving the brand tone of the current site.
- Prioritize readability and discovery.

Brand and mood:

- Confident technical editorial tone.
- Professional and clean, with subtle personality.
- Not playful or cartoonish in UI chrome.

Visual direction:

- Bright, modern editorial interface.
- Strong typography hierarchy and generous spacing.
- Use card-based content modules for discovery sections.
- Use a restrained accent color system inspired by the current site: deep slate neutrals plus one energetic blue/cyan accent.

Typography:

- Heading style: expressive editorial serif or similarly distinctive display style.
- Body style: highly legible sans-serif.
- Clear scale difference between title, subtitle, metadata, and body copy.

Layout behavior:

- Respect desktop vs mobile framing from each prompt.
- Keep module order exactly as requested.
- Maintain clear gutters and alignment rhythm.

Content behavior:

- Use realistic placeholder copy related to SFCC topics.
- Keep text concise and legible.
- Include metadata patterns (date, reading time, topic).

Quality bar:

- This is a polished visual design concept, not a wireframe.
- Use realistic UI details such as spacing, rounded corners, shadows, and subtle separators.
- Keep it implementation-feasible for a modern Hugo theme.

Do not:

- Do not change section order from the prompt.
- Do not add random extra modules not requested.
- Do not use dark-only aesthetic unless explicitly asked.
- Do not create an app dashboard style; this is a content-first blog.

Output:

- Generate one single page mockup image.
- Keep full page visible and centered.
