# Image Prompt Pack

Date: 2026-03-08

This folder contains copy-paste prompts for an image model such as Gemini to generate low-fidelity wireframe images for each page type.

Reference sources:

- `analysis/design/low-fi-wireframes-2026-03-08.md`
- `analysis/documentation/checklists/ui-implementation-checklist-2026-03-08.md`

## How to use

1. Start with `00-shared-style-and-constraints.md`.
2. Then run one page-type file at a time:
   - `01-homepage-prompts.md`
   - `02-archive-prompts.md`
   - `03-article-prompts.md`
3. Generate one desktop and one mobile image for each page type.
4. Save resulting images under:

- `analysis/design/generated-images/`

## Suggested output filenames

- `homepage-desktop-wireframe.png`
- `homepage-mobile-wireframe.png`
- `archive-desktop-wireframe.png`
- `archive-mobile-wireframe.png`
- `article-desktop-wireframe.png`
- `article-mobile-wireframe.png`
