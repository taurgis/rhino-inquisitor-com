# Homepage Prompts

Use together with `00-shared-style-and-constraints.md`.

## Prompt A - Homepage Desktop

Create a low-fidelity desktop wireframe for a technical blog homepage.

Required structure from top to bottom:

1. Top header bar with placeholders: Logo, Blog, Topics, About, Search, Start Reading CTA.
2. Hero row with short intro text area on the left and a featured visual placeholder on the right.
3. Hero action row with two buttons: Start Reading and Browse Topics.
4. Three discovery lanes in one row:
   - Lane A: Featured long-form article card (title, excerpt, metadata strip).
   - Lane B: Recent posts lane with three compact cards.
   - Lane C: Topic hubs lane with topic chips and count indicators.
5. Compact active projects rail with 2-3 list items.
6. Newsletter or subscribe strip.
7. Footer block.

Layout guidance:

- Clear gutters.
- Consistent card spacing.
- Strong hierarchy for hero and discovery lanes.
- Label each major region with short wireframe text.

Image framing:

- Desktop browser-like frame, landscape orientation.

## Prompt B - Homepage Mobile

Create a low-fidelity mobile wireframe for the same blog homepage.

Required structure from top to bottom:

1. Mobile header with Logo, Menu icon, Search icon.
2. Hero intro block with two stacked CTA buttons.
3. Featured long-form card.
4. Recent posts section with 3 stacked compact cards.
5. Topics chip section in horizontal style.
6. Compact projects list.
7. Newsletter strip.
8. Footer block.

Layout guidance:

- Single column.
- Clear vertical rhythm.
- Keep section labels short and readable.

Image framing:

- Smartphone portrait frame.
