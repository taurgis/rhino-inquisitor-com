# Shared Style And Constraints Prompt

Paste this before any page-specific prompt:

Create a low-fidelity web wireframe image.

Style constraints:

- Monochrome grayscale only.
- White background.
- Black and dark gray boxes and lines.
- No gradients.
- No realistic photography.
- No decorative illustration.
- No brand logos.
- Keep text minimal and legible, using simple labels only.
- Use clean rectangular blocks with clear spacing.
- Keep visual hierarchy obvious.
- Include headers, sections, cards, and footer as simple wireframe blocks.

Wireframe intent:

- Focus on layout and information architecture.
- Do not style as a finished UI.
- Emphasize readability and content discovery zones.

Output behavior:

- Produce a single image per prompt.
- If prompt says Desktop, use a desktop page mockup framing.
- If prompt says Mobile, use a mobile portrait frame.
- Keep composition centered and fully visible.

Negative constraints:

- Do not generate colorful mockups.
- Do not generate polished product shots.
- Do not add random gibberish text paragraphs.
- Do not omit section labels.
