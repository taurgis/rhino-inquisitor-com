# Archive Prompts

Use together with `00-shared-style-and-constraints.md`.

## Prompt A - Archive Desktop

Create a low-fidelity desktop wireframe for a blog archive page focused on discovery.

Required structure from top to bottom:

1. Top header with Logo, Blog, Topics, About, Search.
2. Archive title and short descriptor line.
3. Search input row with filter and sort controls.
4. Two-column content area:
   - Left filter rail with sections: Topic, Type, Difficulty, Year.
   - Right listing area with multiple article cards in grid or list form.
5. Each card shows placeholders for image, title, metadata row, and short excerpt.
6. Pagination row with Prev, page numbers, Next.
7. Jump-to-year mini control.
8. Footer block.

Layout guidance:

- Make search/filter controls visually distinct.
- Keep left rail narrow and right content wider.
- Use repeating card blocks for discoverability.

Image framing:

- Desktop browser-like frame, landscape orientation.

## Prompt B - Archive Mobile

Create a low-fidelity mobile wireframe for the same blog archive page.

Required structure from top to bottom:

1. Mobile header with Logo, Menu icon, Search icon.
2. Archive title and short descriptor.
3. Search bar.
4. Filter chip row and sort dropdown row.
5. Stacked article cards (at least 3 visible).
6. Pagination row with Prev and Next.
7. Jump-to-year control.
8. Footer block.

Layout guidance:

- Single-column scannable flow.
- Emphasize discoverability controls above cards.
- Keep cards consistent in structure.

Image framing:

- Smartphone portrait frame.
